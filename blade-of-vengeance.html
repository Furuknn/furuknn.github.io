Blade of Vengeance

2D Pixel-Art Sword Fighting Game

About the Project

Blade of Vengeance is a 2D pixel-art sword-fighting game inspired by the sabre discipline of fencing. Combat emphasizes timing, spacing, and rhythm rather than button mashing.

The game features both PvE and PvP modes. Players can experience story-driven encounters or test their reflexes against friends using four distinct fighting styles.

I worked primarily on gameplay programming, combat logic, hit detection, animation timing, and overall game feel using Unity.

What I Learned

  • TECHNICAL FOUNDATION

    I applied core object-oriented programming principles to structure gameplay systems in a scalable way. Animation Events were used to synchronize attacks, hit detection, and visual feedback. I also implemented AI opponents with multiple difficulty levels by adjusting reaction time, aggression, and decision-making logic.

  • TECHNICAL CHALLENGE

    One of the challenges was to animate the characters and swords separately without adding to the costs of production. This was a problem in pixel art, where each animation has to be hand-drawn, and adding animations for each type of sword would have resulted in a monumental task.

    To address this, we broke down character and weapon animations into modular pieces. This enabled reusing character animations across multiple swords. This approach also ensured that character combat remained responsive.

  • DESIGN INSIGHT

    We designed the game around the concept of fast-paced sword combat. We watched fencing tournaments to see how professionals do it. Dashing and drawing sword logic was inspired by this. We adjusted some of the characters' ability cooldowns to balance the game feel.
